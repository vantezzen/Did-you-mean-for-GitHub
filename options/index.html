GitHub access token

You can optionally add a GitHub API access token to this extension.

By providing a token, this extension will use the GitHub GraphQL API to check if the suggested repositories actually exist.

If you do not supply a token, this extension will still work, but it will suggest repositories that don't exist.

Please create a new access token by clicking here. You don't need to give the token any scopes.

Your access token will only be saved in your browser.
